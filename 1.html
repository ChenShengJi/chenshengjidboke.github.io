Git常用命令总结
	1、 下载git 
	2、  然后注册
	       $ git config --global user.name "Your name"	账号
	       $ git config --global user.email "youxiang"	邮箱
	       $ git config --list     			   查看账号密码
	3、进入d盘 
	        cd d:
	4、  进入d盘的文件夹
	        cd wenjianjia
	5、  输入
	        git init
	        在本地新建一个repo,进入一个项目目录,
	        执行git init,会初始化一个repo,并在当前文件夹下创建一个.git文件夹.
	6、  git status
	        查询状态
	        no commits yet 还没有提交到版本库里
	        红色的文件名代表修改过了
	        查询repo的状态.
	        git status -s: -s表示short,
	        -s的输出标记会有两列,第一列是对staging区域而言,
	       第二列是对working目录而言.
	7、 git add js.js
	        git add
	        在提交之前,Git有一个暂存区(staging area),
	        可以放入新添加的文件或者加入新的改动. 
	        commit时提交的改动是上一次加入到staging area中的改动,
	        而不是我们disk上的改动.
	        git add .
	        会递归地添加当前工作目录中的所有文件.
	8.、 再一次输入status
	        文件夹名会变绿色 说明放到了暂存区
	9、  git commit -m "add one file"
	        从暂存区放到了版本区
	10、git log 
	        查看日志
	11、git checkout -b host  <--这是分支名
	        创建一个分支
	12、git checkout master 
	         退出分支
	         返回master
	         在开始 修改js文件 
	         然后 6 7 8 
	14、diff查看工作区 暂存区 版本区
